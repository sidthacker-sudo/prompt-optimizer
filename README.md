# Prompt Optimizer

Chrome extension that helps you write better prompts for ChatGPT and Claude with AI-powered optimization, scoring, and a library of reusable templates.

## Features

- **Smart Prompt Scoring**: Get instant feedback on your prompt quality (0-100 score)
- **AI-Powered Optimization**: Transform basic prompts into clear, effective instructions with one click
- **Prompt Library**: Save, organize, and reuse your best prompts
- **Conversation Chains**: Save multi-step conversation workflows and replay them
- **Next Prompt Suggestions**: AI suggests contextual follow-up prompts
- **Local & Private**: All your data stays on your device

## Installation

1. Download the latest release from the [Chrome Web Store](#) (coming soon)
2. Add your Anthropic API key in Settings (one-time setup)
3. Go to ChatGPT or Claude and start optimizing your prompts!

## Supported Sites

- ChatGPT (chatgpt.com)
- Claude (claude.ai)
